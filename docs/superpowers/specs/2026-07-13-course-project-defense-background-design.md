# 课程项目答辩 PPT 底板设计说明

## 目标

基于用户提供的参考图，制作一份 16:9 PowerPoint 底板合集，用于课程项目答辩。底板需保持正式、学术、克制的视觉气质，并把主要内容空间留给后续编辑。

## 核心约束

- 中央区域必须保持大面积留白，可安全容纳标题、正文、图表和代码截图。
- 装饰元素集中在顶部约 12% 和底部约 8% 的区域，不侵入主要内容区。
- 使用参考图中的深红、浅灰、白色和少量橙色作为统一色系。
- 从用户提供的参考图片中提取校徽视觉，仅用于本次用户要求的底板。
- 等高线或流线纹理只在上下边缘以低透明度出现，并向中央渐隐。
- 页码、页脚、栏目名称和标题占位均保持可编辑。
- 不复制参考图中的具体章节文字和页码内容。

## 交付内容

生成一份包含 6 张底板的可编辑 PPTX：

1. 封面底板：顶部校徽与红色主视觉，中央完全留白。
2. 目录底板：顶部横向栏目结构，当前栏目用深红色强调。
3. 章节过渡底板：上下装饰形成收束，中部预留章节标题区。
4. 通用内容底板：最克制的上下边缘装饰，中部留白比例最高。
5. 数据图表底板：边缘增加细线和橙色定位标记，中央适合图表。
6. 总结答辩底板：顶部简化、底部红色收束，可用于总结或致谢。

同时导出整套底板的 PNG 拼图预览，便于快速浏览六款差异。

## 视觉系统

- 画布：16:9 宽屏。
- 主色：学术深红，接近参考图的红色。
- 辅色：中性浅灰、暖白。
- 点缀色：小面积橙色，只用于细条、定位点或页码强调。
- 字体：优先使用微软雅黑或思源黑体风格的中文无衬线字体。
- 线条：细、轻、低对比，避免出现模板化边框感。
- 留白安全区：中央约占页面宽度 78%、高度 68%，不放固定装饰。

## 可编辑性与实现

- 使用 PowerPoint 原生文本、色块、线条和基础装饰构建版式。
- 参考图中的校徽以裁剪后的透明或白底图片形式放置。
- 复杂纹理使用单独的低透明度栅格装饰图，避免产生大量难编辑对象。
- 六张底板保持统一母题，但顶部和底部轮廓具有明显差异。

## 验收标准

- 六张页面均为 16:9，视觉风格统一且差异清晰。
- 每张页面中央都有连续、干净的大面积内容空间。
- 固定装饰不遮挡正文常用区域，也不出现越界、裁切或意外重叠。
- 校徽显示清晰，颜色与参考图协调。
- PPTX 可正常打开和编辑；全部页面完成渲染检查。
